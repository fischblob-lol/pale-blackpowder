# pale-blackpowder
Pale-Blackpowder is a free and open-source multipurpose "Jailbreak" tool to enable system R/W on immutable linux distros such as Fedora Silverblue, Origami Linux and others. It is licensed under the GNU-GPL V3.0 License. For more information, please read LICENSE.
